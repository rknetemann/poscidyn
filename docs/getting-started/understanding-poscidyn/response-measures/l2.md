# L2

The `L2` response measure computes the root-mean-square (RMS) value of the response over the provided time samples.

This gives a scalar measure of the response magnitude in the time domain. It does not return a meaningful phase or response frequency.

## Definition

For each mode \(i\), the modal L2 measure is computed as

$$
A_i
=
\sqrt{
\frac{1}{N}
\sum_{k=1}^{N}
x_i(t_k)^2
}
$$

where:

- \(x_i(t_k)\) is the modal response of mode \(i\) at time \(t_k\),
- \(N\) is the number of time samples.

If a mode shape is provided, the total displacement is first reconstructed as

$$
x_{\mathrm{total}}(t_k)
=
\sum_{i=1}^{n_{\mathrm{modes}}}
\phi_i \, x_i(t_k)
$$

and the total L2 measure is then

$$
A_{\mathrm{total}}
=
\sqrt{
\frac{1}{N}
\sum_{k=1}^{N}
x_{\mathrm{total}}(t_k)^2
}
$$

## Returned fields

The result is a dictionary with `"modal"` and `"total"` entries.

For this measure:

- `"amplitude"` contains the RMS value,
- `"phase"` is set to `NaN`,
- `"response_frequency"` is set to `NaN`.

This is because the L2 measure is a time-domain magnitude only and does not encode phase or frequency.
